AI Assistant

A local-first, privacy-respecting, cross-platform desktop assistant. Jarvis-style functionality that runs entirely on your machine.

View Source Code Get Started

Overview

AI Assistant is a cross-platform desktop application that brings Jarvis-style functionality to your computer. Built with a privacy-first architecture, all processing happens locally on your machine - no data is ever sent to external servers.

The system features a modular architecture with strict separation of concerns, making it easy to extend and maintain. Whether you're opening applications, controlling volume, or taking screenshots, every command is processed through a secure, permission-based system.

Architecture

The system follows a clean, layered architecture that separates concerns and enables easy testing and extension. The brain never touches OS-specific code, and adapters translate abstract commands into platform-specific implementations.

Available Commands

Command Description
open <app> Open an application by name
open file <path> Open a file with its default application
open folder <path> Open a folder in file manager
volume <0-100> Set system volume level
mute / unmute Mute or unmute system audio
screenshot Take and save a screenshot
lock Lock the screen
shutdown / restart System power options (with confirmation)
time / date Get current time or date
battery Check battery status
safe mode Enable read-only safe mode
help Show all available commands

Getting Started

Getting up and running takes just a few commands:

# Clone the repository
git clone https://github.com/ShriOg/anand-dev.tech.git
cd anand-dev.tech/projects/ai-assistant

# Run the assistant
python main.py

# Or run a single command
python main.py --command "time"

# Start in safe mode
python main.py --safe-mode
