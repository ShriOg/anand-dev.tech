Spatial Hashing for Collision Detection

Particle system was dying at 300+ particles due to O(n¬≤) collision checks. Implemented spatial hashing: divide space into grid cells, only check particles in same/adjacent cells. Went from 15fps to 60fps with 500 particles. Key insight: reduce the search space before optimizing the search.

MediaPipe Hand Tracking Limitations

MediaPipe tracks at ~30fps but my render loop runs at 60fps. Solution: interpolate hand positions between frames. Track velocity and predict position. Added configurable smoothing factor to reduce jitter. Hand tracking is good, but the gap between tracking fps and render fps matters.

Premature Optimization

2024-11

Spent 2 weeks building a custom caching layer for my API before I had any users. Added Redis, wrote invalidation logic, handled edge cases. Then realized the app got 10 requests per day. A simple in-memory cache would have been fine. Built for scale I didn't have.

Lesson Make it work, then make it fast. Measure before optimizing. "Premature optimization is the root of all evil" isn't just a quote.

Ignored Error Boundaries

2024-09

React app crashed on production because one component threw an error and I had no error boundaries. Entire app went white. Users thought the site was down. The actual bug was a null check in a rarely-used feature.

Lesson Always wrap major sections in error boundaries. Fail gracefully. One broken component shouldn't take down the entire app.

No Migration Strategy

2024-07

Changed database schema without proper migration plan. Had to manually fix production data at 2am. Some users lost settings. Rushed the fix and introduced another bug. Cascading failures from one careless change.

Lesson Never modify production schemas without a tested migration script and rollback plan. Always backup before migrations. 2am debugging is never worth saving 30 minutes of planning.

Feature Creep

2024-05

Started building a simple note-taking app. Added tags. Then folders. Then search. Then collaboration. Then version history. Then export options. 6 months later, I had an unfinished, bloated mess that did nothing well. Never shipped.

Lesson Ship the smallest useful thing first. Everything else is an iteration. A finished simple app beats an unfinished complex one every time.

Trusted User Input

2024-03

Built a form that passed user input directly to an API without sanitization. Someone submitted a script tag in a name field. Fortunately caught in code review before production. But I should have known better.

Lesson Never trust user input. Sanitize everything. Validate on client and server. Use parameterized queries. Security isn't optional.
