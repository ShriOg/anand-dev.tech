Hand Gesture Mouse Control

Computer Vision · Python · 2024

View Source Code →

The Problem

Traditional mouse input requires constant physical contact with hardware, which can lead to repetitive strain injuries during extended computer use. For users with mobility limitations, standard input devices present accessibility barriers.

I wanted to explore whether computer vision could provide a natural, touchless alternative for controlling a computer cursor.

The Approach

The solution uses MediaPipe's hand tracking model to detect and track hand landmarks in real-time through a webcam feed. By mapping specific finger positions and gestures to mouse actions, the system creates an intuitive control interface.

Key technical decisions included implementing smoothing algorithms to reduce cursor jitter, establishing clear gesture thresholds for reliable click detection, and optimizing the processing pipeline for low-latency response.

Technical Implementation

The system processes video frames through OpenCV, extracts hand landmarks using MediaPipe, and translates spatial coordinates to screen positions. Index finger position controls cursor movement, while a pinch gesture (thumb to index finger) triggers click events.

To achieve smooth cursor movement, I implemented an exponential moving average filter that balances responsiveness with stability. The gesture detection uses distance thresholds with hysteresis to prevent unintended clicks.

The Outcome

The final implementation achieves reliable hand tracking at 30+ fps on standard hardware, with cursor movement that feels natural and responsive. The click detection maintains high accuracy while avoiding false positives during normal hand movements.

This project deepened my understanding of real-time computer vision systems, signal processing for human-computer interaction, and the importance of thoughtful UX in gesture-based interfaces.
